# CodeBaseNodeJS
* How to use:
- git clone
- yarn install
- yarn start
* Concept:
- RESTful API backend for NodeJS
- Basic features: singUp, signIn, setAvatar
- Keywords: Bcrypt (hashPWD), JWT (Authorization), Multer (Upload file)
* Call APIs by 1 in 2 ways below: 
- Use Postman
- Use cURL
* PS: Please change your database MONGO_URI
